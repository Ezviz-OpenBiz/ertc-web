# ertc-web sdk

[![Github](https://img.shields.io/badge/Github-Repo-blue?style=flat&logo=github)](https://github.com/Ezviz-OpenBiz/ertc-web-demo)

## 浏览器支持

ERTC Web SDK 基于 WebRTC 实现，目前在浏览器各端的支持如下：
| | chrome | firefox | edge | safari | opera |
| -------------------- | ---------------------------------------: | :----------------------------------------------------------------------: | ------- | ---| ----|
| windows | 72 | 80 | 80	 | - | 90 |
| mac | 72 | 99 | 80	 | 14.1.1 | 90 |

## 项目引入

```
import ERTC from 'ertc-web'

const ertc = new ERTC()
```
